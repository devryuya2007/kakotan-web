# 単語帳アプリ（サンプル）

このリポジトリには、簡単な単語帳アプリのサンプルコードが含まれています。ブラウザで `public/index.html` を開くとホーム画面が表示されます。

公開用ファイルは `public` ディレクトリに配置しています。

## 使い方
1. このリポジトリをクローン、またはダウンロードします。
2. `public/index.html` をブラウザで開きます。
3. ホーム画面で「テスト」を選ぶと単語テスト画面(`test.html`)が開きます。
4. テスト画面では表示された英単語に対して「答えを見る」を押すと日本語訳が表示されます。
5. 「次へ」を押すと次の単語に進みます。最後まで進むと最初に戻ります。
6. ホーム画面の「成績」は成績ページ(`grades.html`)を表示します（現在はサンプルのみ）。
