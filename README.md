# 単語帳アプリ

このリポジトリには、シンプルな単語帳アプリのサンプルが入っています。公開用ファイルは `public` ディレクトリに置いてあります。

## 使い方
1. リポジトリをクローンまたはダウンロードします。
2. ブラウザで `public/index.html` を開きます。
3. ホーム画面のボタンからテスト画面(`test.html`)や成績画面(`grades.html`)へ移動します。
4. テスト画面では「クイズを始める」を押して 20 問の英単語クイズに挑戦できます。
5. 単語一覧ボタンでは登録されている単語を確認できます。
6. 成績ページは現在準備中です。

はじめての方でもコードを読みやすいよう、日本語のコメントを多く入れています。気軽にカスタマイズしてみてください。
