# 単語帳アプリ（サンプル）

このリポジトリには、簡単な単語帳アプリのサンプルコードが含まれています。ブラウザで `public/index.html` を開くことで動作を確認できます。

## 使い方
1. このリポジトリをクローン、またはダウンロードします。
2. `public/index.html` をブラウザで開きます。
3. 画面に表示された英単語に対して「答えを見る」を押すと日本語訳が表示されます。
4. 「次へ」を押すと次の単語に進みます。最後まで進むと最初に戻ります。
